## Installing Wayland rofi from source on Debian/Ubuntu
Meson must be used. The following packages are required:

```
libpango1.0-dev libcairo2-dev libgdk-pixbuf-2.0-dev libstartup-notification0-dev libxkbcommon-dev libxkbcommon-x11-dev libxcb1-dev libxcb-util-dev libxcb-ewmh-dev libxcb-cursor-dev wayland-protocols meson libxcb-icccm4-dev libxcb-randr0-dev libxcb-xinerama0-dev libwayland-client0 libwayland-dev
```
